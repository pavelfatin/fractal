Fractal README
============

Fractal is a [Mandelbrot set](http://en.wikipedia.org/wiki/Mandelbrot_set "Mandelbrot set - Wikipedia") rendering application (GPU / CPU).

Features:

* OpenCL / Multithreaded / Singlethreaded renderer.
* Incremental rendering.
* Smooth scrolling.
